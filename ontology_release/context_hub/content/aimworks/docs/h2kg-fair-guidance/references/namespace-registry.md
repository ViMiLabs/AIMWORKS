# Namespace Registry

Approved namespaces for H2KG release work include:

- `h2kg`: `https://w3id.org/h2kg/hydrogen-ontology#`
- `owl`: `http://www.w3.org/2002/07/owl#`
- `rdf`: `http://www.w3.org/1999/02/22-rdf-syntax-ns#`
- `rdfs`: `http://www.w3.org/2000/01/rdf-schema#`
- `skos`: `http://www.w3.org/2004/02/skos/core#`
- `dcterms`: `http://purl.org/dc/terms/`
- `prov`: `http://www.w3.org/ns/prov#`
- `vann`: `http://purl.org/vocab/vann/`
- `qudt`: `http://qudt.org/schema/qudt/`
- `unit`: `http://qudt.org/vocab/unit/`
- `quantitykind`: `http://qudt.org/vocab/quantitykind/`

Use the local `h2kg` namespace for PEMFC catalyst-layer application terms that remain part of the ontology.

Reuse the external namespaces above rather than minting duplicates for common metadata and quantity semantics.
