# Release Checklist

Before publishing, confirm:

- inspection report exists
- split report exists
- alignment report exists
- validation report exists
- FAIR readiness report exists
- release readiness report exists
- schema outputs exist in Turtle and JSON-LD
- example module exists
- mapping review CSV exists
- annotation draft review files exist if drafting was used
- static docs site exists
- w3id artifacts exist
- CITATION and Zenodo metadata are present
