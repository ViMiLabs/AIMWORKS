# Content Inventory

The H2KG Context Hub pack contains these primary entries:

- `h2kg-overview`
- `h2kg-ontology-model`
- `h2kg-classes`
- `h2kg-properties`
- `h2kg-alignments`
- `h2kg-iri-policy`
- `h2kg-examples`
- `h2kg-validation`
- `h2kg-release-process`
- `h2kg-fair-guidance`

Reference material is attached under `references/` folders for the entries that need compact supporting guidance such as:

- modeling patterns
- anti-patterns
- mapping rules
- annotation guidance
- namespace registry
- release checklist

Nothing in this pack is meant to replace the generated HTML docs. The purpose is targeted retrieval by agents.
