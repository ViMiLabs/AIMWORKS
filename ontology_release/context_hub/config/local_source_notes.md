# Local Source Notes

This repository does not hard-bind itself to a specific Context Hub config schema because upstream CLI conventions can evolve.

The content layout follows the documented author/docs/entry pattern so it can be adapted into a local source build with minimal glue:

- `content/aimworks/docs/<entry>/DOC.md`
- optional `references/` under an entry for compact supporting materials

Use this folder as the canonical source when configuring a local H2KG Context Hub registry.

Recommended local source label:

- `h2kg-local`

Recommended source description:

- `Local Context Hub pack for the H2KG PEMFC Catalyst Layer Application Ontology`

Recommended retrieval scope:

- ontology scope
- class and property usage
- alignment policy
- identifier policy
- validation and release process
