# Publication Layout

This directory is intended to be deployed directly as the public static publication root.

## Reference IRIs

- `Ontology IRI`: `https://w3id.org/h2kg/hydrogen-ontology` - Canonical ontology IRI with HTML/RDF content negotiation.
- `Asserted Source`: `https://w3id.org/h2kg/hydrogen-ontology/source` - Latest asserted source ontology serialization.
- `Inferred Source`: `https://w3id.org/h2kg/hydrogen-ontology/inferred` - Latest inferred ontology serialization.
- `Latest Release`: `https://w3id.org/h2kg/hydrogen-ontology/latest` - Stable alias for the latest asserted release.
- `JSON-LD Context`: `https://w3id.org/h2kg/hydrogen-ontology/context` - Stable JSON-LD context endpoint.
- `Versioned Release`: `https://w3id.org/h2kg/hydrogen-ontology/2026.3.0` - Version-pinned asserted release.
- `Versioned Inferred`: `https://w3id.org/h2kg/hydrogen-ontology/2026.3.0/inferred` - Version-pinned inferred release.
