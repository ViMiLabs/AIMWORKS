Ontology Overview

What H2KG is, what it models, and which public release resources are available.

H2KG is an EMMO-aligned application ontology for low-Pt PEMFC cathode catalyst-layer experiments, measurements, materials, provenance, and curated domain vocabulary. It is published as a conservative release package that separates schema, curated reference terms, and example or data-like content.
