How to Import

Practical import and programmatic access guidance for the PEMFC ontology release modules.

Import guidance

  • Import the asserted schema module when you need the curated ontology structure without example individuals.
  • Use the inferred serialization for exploration, quality review, or graph analytics, not as a substitute for editorial curation.
  • Keep the controlled vocabulary and example modules separate unless your downstream use case explicitly requires them.
