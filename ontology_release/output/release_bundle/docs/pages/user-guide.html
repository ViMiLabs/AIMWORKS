User Guide

Run the pipeline from the ontology_release directory. The default release command performs inspection, split, alignment, enrichment, validation, documentation generation, w3id artifact generation, FAIR scoring, and release bundling.

python -m aimworks_ontology_release.cli release --input input/current_ontology.jsonld

For stepwise review:

  • inspect writes ontology diagnostics and blockers.
  • split separates schema, vocabulary-like resources, and example/data-like content.
  • map generates conservative alignment proposals and a CSV review sheet.
  • annotate creates reviewable annotation drafts, with optional LLM support.
  • validate runs metadata, namespace, mapping, and SHACL checks.
