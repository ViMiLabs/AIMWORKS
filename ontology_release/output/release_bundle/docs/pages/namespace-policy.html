Namespace Policy

Stable IRI and namespace policy for the active H2KG publication profile.

Active release profile

  • Active profile: stable_hash
  • Ontology IRI: https://w3id.org/h2kg/hydrogen-ontology
  • Term namespace: https://w3id.org/h2kg/hydrogen-ontology#
  • Namespace mode: hash
  • Preserve legacy IRIs: True
  • Allow namespace migration: False

Policy notes

  • The stable release profile preserves the current ontology IRI and hash-style term namespace for backward compatibility.
  • Future slash-namespace publication is supported through configuration, but it is not the active default.
  • Namespace migration must produce redirect templates and a migration map rather than silently rewriting local identifiers.
  • Version-specific release IRIs are published alongside the canonical ontology IRI and latest alias.
