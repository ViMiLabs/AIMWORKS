Interactive ontology querying

Run Queries Directly in the Browser

Use the interactive query console to execute SPARQL directly in the browser against the selected ontology modules. H2KG benefits most from queries that make schema, vocabulary, mappings, examples, and release metadata explicit.

  • Start with the term finder or a preset, then refine the query in the console and rerun.
  • Keep sources explicit; asserted schema, controlled vocabulary, alignments, inferred export, and examples answer different questions.
  • Use `LIMIT` in exploratory queries to keep browser execution responsive for end users.
  • Pair each query preset with a domain interpretation so non-ontology users understand the result context.

Competency question presets

Ready-Made Queries

CQ1 SPARQL preset

Which measurement classes in H2KG record instrument usage and produce data outputs?

Use the schema module to traverse local measurement classes and the object properties that relate measurements to instruments and outputs.

PREFIX h2kg: 
PREFIX owl: 
PREFIX rdfs: 

SELECT ?measurement ?measurementLabel ?property ?propertyLabel
WHERE {
  ?measurement a owl:Class ;
    rdfs:label ?measurementLabel .
  FILTER(CONTAINS(LCASE(STR(?measurementLabel)), "measurement"))
  ?property rdfs:domain ?measurement ;
    rdfs:label ?propertyLabel .
}
ORDER BY ?measurementLabel ?propertyLabel
CQ2 SPARQL preset

Which local terms are aligned to EMMO, ECHO, QUDT, ChEBI, or PROV-O?

Query the alignments graph to review proposed reuse and local terms that remain intentionally specific to H2KG.

PREFIX skos: 
PREFIX owl: 
PREFIX rdfs: 

SELECT ?local ?relation ?external
WHERE {
  VALUES ?relation { owl:equivalentClass owl:equivalentProperty rdfs:subClassOf rdfs:subPropertyOf skos:exactMatch skos:closeMatch }
  ?local ?relation ?external .
  FILTER(STRSTARTS(STR(?local), "https://w3id.org/h2kg/hydrogen-ontology#"))
}
ORDER BY ?local ?relation

CQ4 SPARQL preset

Which local schema terms are currently marked as deprecated?

Use this query to identify terms that remain resolvable but should no longer be used for new annotations.

PREFIX owl: 
PREFIX rdfs: 

SELECT ?term ?label ?type
WHERE {
  VALUES ?type { owl:Class owl:ObjectProperty owl:DatatypeProperty owl:AnnotationProperty }
  ?term a ?type ;
    owl:deprecated true .
  OPTIONAL { ?term rdfs:label ?label }
  FILTER(STRSTARTS(STR(?term), "https://w3id.org/h2kg/hydrogen-ontology#"))
}
ORDER BY ?type ?label
CQ5 SPARQL preset

Which local properties declare domain and range constraints?

This query helps reviewers inspect modeling constraints and detect properties that may need stronger domain or range curation.

PREFIX owl: 
PREFIX rdfs: 

SELECT ?property ?propertyLabel ?kind ?domain ?range
WHERE {
  VALUES ?kind { owl:ObjectProperty owl:DatatypeProperty }
  ?property a ?kind .
  OPTIONAL { ?property rdfs:label ?propertyLabel }
  OPTIONAL { ?property rdfs:domain ?domain }
  OPTIONAL { ?property rdfs:range ?range }
  FILTER(STRSTARTS(STR(?property), "https://w3id.org/h2kg/hydrogen-ontology#"))
}
ORDER BY ?kind ?propertyLabel

CQ8 SPARQL preset

Which local classes and properties still miss labels or textual definitions?

This is a quality-control query for publication editing, focused on local schema terms.

PREFIX owl: 
PREFIX rdfs: 
PREFIX skos: 

SELECT ?term ?type
WHERE {
  VALUES ?type { owl:Class owl:ObjectProperty owl:DatatypeProperty owl:AnnotationProperty }
  ?term a ?type .
  FILTER(STRSTARTS(STR(?term), "https://w3id.org/h2kg/hydrogen-ontology#"))
  FILTER NOT EXISTS { ?term rdfs:label ?label }
  FILTER NOT EXISTS { ?term rdfs:comment ?comment }
  FILTER NOT EXISTS { ?term skos:definition ?definition }
}
ORDER BY ?type ?term
