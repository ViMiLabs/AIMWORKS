Release Story

Source inspection and separation

Inspect the mixed ontology source, classify resources conservatively, and separate schema, controlled vocabulary, and example or data-like resources.

Alignment and metadata enrichment

Propose conservative mappings, preserve local PEMFC terms, and enrich ontology metadata for a FAIR release.

Validation, FAIR checks, and publication

Validate the release with SHACL and metadata rules, compute FAIR readiness signals, and publish HTML, RDF, and w3id artifacts together.

Keep the HTML portal for browsing and use the machine-readable release files for ontology loading, scripting, and downstream graph workflows.

Merged asserted release source/asserted.ttl BattINFO-inspired merged asserted release across local schema, controlled vocabulary, and reviewed mappings.

Merged asserted release (RDF/XML) source/asserted.rdf RDF/XML export of the merged asserted release.
Inferred release inferred/ontology.ttl Inferred schema and mapping export from the current publication tree.
