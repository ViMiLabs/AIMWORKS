PEMFC Ontology Profile

Ontology for representing low-Pt PEM fuel-cell cathode catalyst layer experiments, focusing on ionomer/solvent effects on ECSA, high-frequency resistance, and oxygen-transport metrics over realistic operating conditions.

Use Cases
