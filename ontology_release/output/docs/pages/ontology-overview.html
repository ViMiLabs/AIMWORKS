H2KG PEMFC Catalyst Layer Application Ontology is an EMMO-aligned application ontology rather than a broad hydrogen-economy ontology.

Its primary scope is PEMFC cathode catalyst-layer experiments, materials, processes, measurements, and provenance.

The current release snapshot contains 35 schema terms and preserves the original h2kg identifiers by default.
