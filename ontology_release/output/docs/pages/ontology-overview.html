H2KG Ontology

Ontology Overview

EMMO-aligned ontology for hydrogen electrochemical systems, experiments, materials, measurements, and FAIR metadata

Ontology release

Developed through DECODE, the Helmholtz Metadata Collaboration (HMC), and AIMWORKS to support FAIR ontology publication and reusable metadata infrastructure for hydrogen electrochemical systems.

H2KG Application Ontology for Hydrogen Electrochemical Systems is an EMMO-aligned application ontology rather than a broad hydrogen-economy ontology.

Its primary scope is hydrogen electrochemical systems, including PEMFC and PEMWE experiments, materials, processes, measurements, data, and provenance.

The current release snapshot contains 32 schema terms and preserves the original h2kg identifiers by default.

CORE ontology IRI: https://w3id.org/h2kg/hydrogen-ontology

PEMFC ontology IRI: https://w3id.org/h2kg/pemfc/hydrogen-ontology

PEMWE ontology IRI: https://w3id.org/h2kg/pemwe/hydrogen-ontology

Release Architecture

AIMWORKS remains the domain-specific release and documentation layer, while ODK is introduced as a nested standard ontology engineering and QC backend.

PEMFC and PEMWE profile modules remain AIMWORKS outputs in shadow mode, and ODK does not change public H2KG IRIs unless a later promotion step is explicitly approved.

Semantic Division of Labor

HDO is the primary alignment source for Helmholtz-community digital-data concepts such as metadata, identifiers, digital objects, schemas, and validation.

EMMO / ECHO remain the scientific and electrochemistry anchors, QUDT covers quantities and units, ChEBI covers chemistry, and PROV-O / DCTERMS remain publication metadata anchors.
