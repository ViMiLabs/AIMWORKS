The H2KG PEMFC Catalyst Layer Application Ontology is published conservatively. The recommended maintenance workflow is inspect, split, review mappings, enrich metadata, validate, build docs, then publish.

Maintain local PEMFC-specific terms under the h2kg namespace unless a future migration policy is approved and redirect artifacts are prepared.

Use QUDT for quantity kinds and units, ChEBI for chemicals when resolvable, and PROV-O plus DCTERMS for release metadata and provenance.
