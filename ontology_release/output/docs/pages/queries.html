Competency-Question-Driven Querying

Start with competency-question-driven SPARQL examples before exposing a full public endpoint. H2KG benefits most from queries that demonstrate separation of schema, vocabulary, examples, mappings, and release metadata.

  • Publish ready-made example queries together with short explanations of what each result means for ontology users.
  • When a public endpoint is added, keep it read-only and document which graphs are queryable.
  • Pair every query example with a narrative domain use case so the site remains understandable to non-ontology specialists.
CQ1 SPARQL example

Which measurement classes in H2KG record instrument usage and produce data outputs?

Use the schema module to traverse local measurement classes and the object properties that relate measurements to instruments and outputs.

PREFIX h2kg: 
PREFIX owl: 
PREFIX rdfs: 

SELECT ?measurement ?measurementLabel ?property ?propertyLabel
WHERE {
  ?measurement a owl:Class ;
    rdfs:label ?measurementLabel .
  FILTER(CONTAINS(LCASE(STR(?measurementLabel)), "measurement"))
  ?property rdfs:domain ?measurement ;
    rdfs:label ?propertyLabel .
}
ORDER BY ?measurementLabel ?propertyLabel
CQ2 SPARQL example

Which local terms are aligned to EMMO, ECHO, QUDT, ChEBI, or PROV-O?

Query the alignments graph to review proposed reuse and local terms that remain intentionally specific to H2KG.

PREFIX skos: 
PREFIX owl: 
PREFIX rdfs: 

SELECT ?local ?relation ?external
WHERE {
  VALUES ?relation { owl:equivalentClass owl:equivalentProperty rdfs:subClassOf rdfs:subPropertyOf skos:exactMatch skos:closeMatch }
  ?local ?relation ?external .
  FILTER(STRSTARTS(STR(?local), "https://w3id.org/h2kg/hydrogen-ontology#"))
}
ORDER BY ?local ?relation
CQ3 SPARQL example

Which controlled vocabulary terms were kept local because no safe external mapping was found?

Review the mapping review output for local vocabulary terms without a target IRI and use that list to prioritize curation.

PREFIX h2kg: 
PREFIX rdfs: 

SELECT ?term ?label
WHERE {
  ?term rdfs:label ?label .
  FILTER(STRSTARTS(STR(?term), "https://w3id.org/h2kg/hydrogen-ontology#"))
}
ORDER BY ?label
